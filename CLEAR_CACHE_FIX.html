Manual Steps:

  1. Open your app at http://localhost:5000

  3. Go to Console tab

localStorage.clear();
sessionStorage.clear();
location.reload(true);

What This Does:

  • Clears React Query cache
  • Clears localStorage
  • Clears sessionStorage
  • Forces a fresh reload from server

Long-term Fix (Optional):

To prevent this in future, we can change staleTime: Infinity to a reasonable value like 5 minutes:

staleTime: 1000 * 60 * 5, // 5 minutes
